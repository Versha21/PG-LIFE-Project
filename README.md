# PG LIFE Project

## 📌 Overview
PG LIFE is a web-based PG/Hostel listing platform where users can view available PGs, check details, and explore accommodation options.

---

## 🚀 Features
- User registration and login system
- PG listing by city
- PG details page
- Database integration (MySQL)
- Responsive UI (HTML, CSS, JavaScript, PHP)

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP / Hosting)

---

## 📁 Project Setup (Local)
1. Install XAMPP
2. Copy project folder into `htdocs`
3. Start Apache & MySQL
4. Import database (`.sql file`) in phpMyAdmin
5. Open browser:http://pglife2026.infinityfreeapp.com/
6. ---

## 🌐 Deployment
- Upload project to hosting (InfinityFree / Render / etc.)
- Update database credentials in `database_connect.php`

---

## ⚠️ Notes
- Ensure database connection is properly configured
- Replace localhost credentials when deploying live
